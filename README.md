## Overview

This repository contains the complete analysis code, data processing pipeline, and trained model objects for the study:

> **Rahman MR, Rose JI, Rahman MR. Machine Learning-Based Multi-Class Classification of Beta-Thalassemia Trait and Hemoglobin E Carrier Status Using HPLC and Complete Blood Count Parameters. *PLOS ONE*. 2026. (Under review)**

The study trains and evaluates six supervised machine learning classifiers (Logistic Regression, K-Nearest Neighbors, Support Vector Machine, Random Forest, XGBoost, and LightGBM) on a publicly available HPLC-based thalassemia screening dataset of 12,885 individuals from West Bengal, India. The task is a three-class classification problem: Normal, Beta-Thalassemia Trait (BTT) carrier, and Hemoglobin E (HbE) carrier. SHAP-based interpretability analysis is included.

## Key Results

| Model | Accuracy (%) | F1-Macro (%) | AUC-ROC |
|---|---|---|---|
| Random Forest | 100.00 | 100.00 | 1.000 |
| LightGBM | 100.00 | 100.00 | 1.000 |
| XGBoost | 99.96 | 99.85 | 1.000 |
| SVM | 99.42 | 97.73 | 0.999 |
| Logistic Regression | 99.81 | 99.24 | 1.000 |
| KNN | 95.77 | 87.61 | 0.977 |

**CBC-only XGBoost (no HPLC data): Accuracy 92.01%, AUC-ROC 0.914**

**10-fold cross-validation (XGBoost, full features): Accuracy 99.98% ± 0.05%**

## Data Source

The dataset used in this study is publicly available on Kaggle:

**Ghosh A. HPLC-based thalassemia screening data. Kaggle; 2025.**
URL: https://www.kaggle.com/datasets/abhraghoshcmc/hplc-based-thalassemia-screening-data

To reproduce the analysis, download the dataset from the Kaggle link above and place `HPLC data.csv` in the `data/` directory. The dataset is not redistributed in this repository.

## Reproducibility

All analyses use a fixed random seed of 42 throughout. The full pipeline is reproducible by running the notebooks in order (01 through 05). Python package versions are pinned in `requirements.txt`.
