# Data Directory

## Source Dataset

The analysis uses a publicly available HPLC-based thalassemia screening dataset:

**Citation:** Ghosh A. HPLC-based thalassemia screening data [dataset]. Kaggle; 2025.
**URL:** https://www.kaggle.com/datasets/abhraghoshcmc/hplc-based-thalassemia-screening-data
**Access:** Free download (Kaggle account required)

### Download Instructions

1. Create a free Kaggle account at https://www.kaggle.com
2. Navigate to the dataset URL above
3. Click "Download" to obtain `archive.zip`
4. Extract `HPLC data.csv` from the archive
5. Place `HPLC data.csv` in this `data/` directory

The dataset is not redistributed in this repository in compliance with Kaggle terms of use.

## Data Dictionary

The source file `HPLC data.csv` contains the following columns used in this study:

| Column | Type | Unit | Description |
|---|---|---|---|
| Age | String | Years (parsed) | Age of the individual at screening (e.g., "21 Yrs 4 months") |
| Gender | String | — | Sex of participant (Male / Female) |
| Present District | String | — | District of residence (not used in modelling) |
| HbA0 | Float | % | Haemoglobin A0 fraction from HPLC |
| HbA2 | Float | % | Haemoglobin A2 fraction from HPLC; elevated in BTT (>3.5%) and HbE carriers |
| HbF | Float | % | Foetal haemoglobin fraction from HPLC |
| RBC | Float | 10^6/µL | Red blood cell count from CBC |
| HB | Float | g/dL | Haemoglobin concentration from CBC |
| MCV | Float | fL | Mean corpuscular volume from CBC |
| MCH | Float | pg | Mean corpuscular haemoglobin from CBC |
| MCHC | Float | g/dL | Mean corpuscular haemoglobin concentration from CBC |
| RDWcv | Float | % | Red cell distribution width (coefficient of variation) from CBC |
| Diagnosis | String | — | HPLC-based diagnostic class label (13 categories in original dataset) |

### Diagnostic Classes Used in This Study

From the 13 original diagnostic categories, the following three were retained:

| Original Label | Analytical Label | Count |
|---|---|---|
| Normal | Normal | 11,777 |
| Beta Thalassaemia carrier | BTT Carrier | 562 |
| HbE carrier | HbE Carrier | 546 |
| **Total** | | **12,885** |

### Feature Engineering Applied

- **Age:** parsed from string to integer (year component extracted via regex)
- **Gender:** binary-encoded (Female = 0, Male = 1)
- **Diagnosis:** whitespace-stripped before class selection

### Data Quality Notes

Two physiologically impossible MCHC values were identified in the BTT group:
- MCHC = 316.0 g/dL (impossible; biological ceiling ~37 g/dL)
- MCHC = 303.0 g/dL (impossible)

These two records are excluded from the descriptive MCHC statistic for the BTT class (Table 2 of the manuscript, footnote) but are retained in all model training runs, as tree-based models are robust to such outliers.

## Processed Data

The `data/processed/` subdirectory is generated by running Notebook 02 (preprocessing). It will contain:

- `X_full_train.npy` — Training feature matrix, full feature set, post-SMOTE
- `X_full_test.npy` — Test feature matrix, full feature set
- `X_cbc_train.npy` — Training feature matrix, CBC-only, post-SMOTE
- `X_cbc_test.npy` — Test feature matrix, CBC-only
- `y_train.npy` — Training labels (post-SMOTE)
- `y_test.npy` — Test labels (natural distribution)
- `label_encoder.pkl` — LabelEncoder object (class mapping: BTT=0, HbE=1, Normal=2)
- `scaler_full.pkl` — StandardScaler fitted on full feature training set
- `scaler_cbc.pkl` — StandardScaler fitted on CBC-only training set
